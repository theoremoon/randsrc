# RANDSRC

Generate random strings that derived by a given BNF

## example

```
$ ./randsrc ./bnf/bnf.txt syntax
     <z>::=<H>   <V>   ''<f>'T'   |'x"''''D' <R>| "'"
 <j> ::=     ""  <t-5>''|<z><O-> "u0"<u><u-->'' 

$ ./randsrc ./bnf/ebnf.txt grammar
c='}';t5={"_"};mI="5_{";V_7="5N_";

```
